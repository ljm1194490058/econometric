我们正招聘以下职位


岗位名称1：研究助理/行政助理/实习生
工作地点：北京
工作性质：全职
最低学历：本科
招聘人数：2-3人
社会福利：-五险一金、绩效工资

职位描述
1. 参与研究咨询项目，能独立完成如收集数据、整理数据、分析数据和撰写报告等研究任务。 
2. 清晰了解客户需求，根据客户需求设计项目方案。 
3. 维护公司运营平台，如公司网站与微信公众平台等。
4. 中英文资料翻译，美化项目报告、汇报材料、运营平台等。
5. 组织、协调、监督公司运营，如组织培训和会议、数据软件销售等。 

任职要求： 
1. 本科及以上学历。 
2. 熟练的 EXCEL/PPT或其他排版技能，有一定的美工基础。 
3. 较强中英文表达能力；
4. 强烈的责任心，工作细致，具有良好的团队合作精神，能够承担一定的工作压力。 


岗位名称2：经济建模顾问 / 经济分析顾问

工作地点：北京
工作性质：全职
最低学历：本科
招聘人数：1人
社会福利：五险一金、绩效工资、股份激励

职位描述
1. 投入产出分析
2. 一般均衡模型分析 
3. 数据处理与分析

任职要求： 
1. 本科学历，有3年相关工作经验，硕士以上学历，需要有相关学习经历或研究；
2. 较强中英文字表达能力；
3. 有一定数据库、数据处理、分析经验。 

4. 有GEMPACK/ R/ GAMS/Python等软件经验的优先。
